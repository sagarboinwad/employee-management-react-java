# Employee Management System (Full Stack)

This is a Full Stack Employee Management System built using React for the frontend, Java (Spring Boot) for the backend, and MySQL as the database.

The project demonstrates CRUD operations and frontend–backend integration using REST APIs.

---

## Tech Stack

Frontend:
- React
- JavaScript
- HTML
- CSS

Backend:
- Java
- Spring Boot

Database:
- MySQL

---

## Features

- Add employee
- View employee list
- Update employee details
- Delete employee
- REST API based backend
- React frontend integration

---

## How to Run the Project

### Backend (Java – Spring Boot)
1. Open the backend project in IntelliJ or Eclipse
2. Configure MySQL database
3. Update database details in `application.properties`
4. Run the Spring Boot application

---

### Frontend (React)
1. Open terminal
2. Go to frontend folder
3. Run the following commands:
   npm install
   npm start

4. Open browser and visit:
   http://localhost:3000


---

## Database Configuration Example

spring.datasource.url=jdbc:mysql://localhost:3306/employeedb
spring.datasource.username=root
spring.datasource.password=your_password
